# Content Type Templates

Standardized structures for all betting content types.

---

## Template Selection

```
Reviewing ONE sportsbook? → Template 1: Review
Comparing MULTIPLE sportsbooks? → Template 2: Comparison
Teaching HOW TO do something? → Template 3: How-To
About betting in a STATE? → Template 4: State Page
```

---

## Template 1: Sportsbook Review

**URL Pattern:** `/sport/betting/[brand]-review.htm`
**Word Count:** 3,500-4,000 words
**T&Cs:** Complete (1 brand)

### Structure
```
H1: [Brand] Sportsbook Review: [Timeless Angle]
├── Last Updated Badge
├── Introduction (100-150w)
├── Comparison Table (3 brands)
├── Quick Answer Box
├── H2: Overview & Background (300-350w)
├── H2: Getting Started (300-350w)
├── H2: Current Welcome Bonus (400-500w)
│   ├── H3: Promo Code
│   ├── H3: Complete T&Cs ← FULL T&Cs HERE
│   └── H3: Is This Good?
├── H2: Mobile App (500-600w)
├── H2: [Signature Feature] (400-500w)
├── H2: Live Betting (400-450w)
├── H2: Betting Markets (400-450w)
├── H2: Odds Quality (350-400w)
├── H2: Banking (400-450w)
├── H2: Customer Support (300-350w)
├── H2: Loyalty Program (300-400w) ← If missing, explain gap
├── H2: vs Competitors (400-500w)
├── H2: Is It Legal? (300-350w)
├── Pros & Cons
├── FAQs (7+)
├── Final Verdict (200-250w)
├── References (5-10)
├── Related Pages (3+)
└── Responsible Gambling
```

### Key Requirements
- User feedback MUST cite sources (App Store, Reddit)
- Complete T&Cs in dedicated section
- If no loyalty program → dedicated H2 explaining gap

## Template 2: Comparison Page

**URL Pattern:** `/sport/betting/best-[category].htm`
**Word Count:** 2,500-3,500 words
**T&Cs:** Complete (ALL brands)

### Structure
```
H1: Best [Category] Sites
├── Last Updated Badge
├── Introduction (100-150w)
├── Comparison Table (5-7 brands) ← BEFORE main content
├── Quick Answer Box
├── H2: Quick Winner Summaries (100w each)
├── H2: [Brand #1] Review (500w)
├── H2: [Brand #2] Review (400w)
├── H2: [Brand #3] Review (350w)
├── H2: [Brand #4] Review (300w)
├── H2: [Brand #5] Review (250w)
├── H2: Complete Bonus Terms ← ALL BRANDS
├── H2: How We Test (400w)
├── H2: Alternative Options (300w)
├── H2: What to Look For (400w)
├── FAQs (5-7)
├── References (5-10)
├── Related Pages (3+)
└── Responsible Gambling
```

### Key Requirements
- T&Cs for EVERY brand (not just top 3)
- Comparison table before main content
- Brand selection from research

## Template 3: How-To Guide

**URL Pattern:** `/sport/betting/how-to-[action].htm`
**Word Count:** 1,500-2,500 words
**T&Cs:** Brief only (if bonuses mentioned)

### Structure
```
H1: How to [Action]
├── Last Updated Badge
├── Introduction (100-150w)
├── Quick Answer Box
├── H2: What You Need (200w)
├── H2: Step-by-Step (800w)
│   ├── Step 1: Choose Sportsbook
│   ├── Step 2: Create Account
│   ├── Step 3: Verify Identity
│   ├── Step 4: Deposit
│   ├── Step 5: Navigate
│   ├── Step 6: Place Bet
│   ├── Step 7: Track
│   └── Step 8: Withdraw
├── H2: Common Mistakes (300w)
├── H2: Pro Tips (300w)
├── H2: Best Sportsbooks (optional, 200w)
├── H2: Next Steps (200w)
├── FAQs (5-7)
├── References (5-10)
├── Related Pages (3+)
└── Responsible Gambling
```

### Key Requirements
- Clear numbered steps
- Brief T&Cs acceptable
- Limit brands to 3

## Template 4: State Page

**URL Pattern:** `/sport/betting/legal-states/[state].htm`
**Word Count:** 2,000-2,800 words
**T&Cs:** Complete (state-licensed operators)

### Structure
```
H1: [State] Sports Betting Sites
├── Last Updated Badge
├── Introduction (100-150w)
├── Quick Answer Box ← Legal status indicator
├── Comparison Table ← ONLY state-licensed operators
├── H2: Is Betting Legal in [State]? (400w)
├── H2: Licensed Operators (500w)
├── H2: Complete Bonus Terms ← State operators only
├── H2: Types of Betting (400w)
├── H2: Where to Bet (400w)
├── H2: [State] Teams (300w)
├── H2: Laws & Regulations (400w)
├── H2: Tax Information (300w)
├── H2: Responsible Gambling (200w)
├── FAQs (5-7, state-specific)
├── References (5-10)
├── Related Pages (3+)
└── Responsible Gambling
```

### Key Requirements
- Legal status from gaming commission
- ONLY state-licensed operators
- State-specific T&Cs
- Tax and regulation info

---

## Universal Requirements

### Every Page Must Have
- [ ] Intro: 100-150 words
- [ ] Quick Answer Box
- [ ] 5-7+ FAQs with schema
- [ ] 5-10 References
- [ ] 3+ Related Pages
- [ ] Responsible Gambling section
- [ ] Affiliate disclosure
- [ ] No dated language in title

### T&Cs by Template

| Template | Required | Version |
|----------|----------|---------|
| 1 (Review) | ✅ Yes | Complete (1 brand) |
| 2 (Comparison) | ✅ Yes | Complete (ALL) |
| 3 (How-To) | ⚠️ If bonuses | Brief only |
| 4 (State) | ✅ Yes | Complete (state ops) |
