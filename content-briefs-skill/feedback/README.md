# Feedback System Documentation

## Quick Start

### I'm a Writer
1. Copy `FEEDBACK-TEMPLATE.md`
2. Rename to `submitted/[page-name]-feedback-[date].md`
3. Fill out sections: Overall Rating, What Worked, What Needs Improvement, Writer Experience
4. Save and notify your manager

### I'm SEO/Manager
1. Copy `FEEDBACK-TEMPLATE.md`
2. Fill out: Keyword Issues, SEO Performance, Actionable Changes
3. Submit to `submitted/` folder

### I'm the Reviewer
1. Check `submitted/` folder weekly
2. Validate each feedback file
3. Move to `validated/` folder
4. Update reference docs based on patterns
5. Move to `applied/` after changes made
6. Update `FEEDBACK-LOG.md`

## Folder Structure

```
feedback/
├── README.md                    # This file
├── FEEDBACK-TEMPLATE.md         # Copy this to submit feedback
├── FEEDBACK-PROCESS.md          # Detailed process documentation
├── FEEDBACK-LOG.md              # Track improvements over time
├── submitted/                   # New feedback goes here
├── validated/                   # Reviewed and approved feedback
└── applied/                     # Feedback that's been integrated
```

---

## Files Explained

### FEEDBACK-TEMPLATE.md
Standardized form for all feedback. Copy this file, rename it, and fill it out.

### FEEDBACK-PROCESS.md
Complete workflow documentation:
- Who submits what type of feedback
- How feedback gets reviewed and validated
- How lessons get extracted and applied
- Integration with brief generation system

### FEEDBACK-LOG.md
Running log of all changes made to the system based on feedback. Includes:
- Individual feedback resolutions
- Metrics tracking (generation time, quality scores, etc.)
- Improvement changelog
- Success stories

## Why This Matters

Every piece of feedback makes the system better for everyone:

- **Writers** get clearer instructions, spend less time on briefs
- **SEO team** gets better keyword research, less cannibalization
- **Project managers** see quality improve and revision cycles decrease
- **Business** gets better-performing content that ranks higher

The system learns from every brief generated.
