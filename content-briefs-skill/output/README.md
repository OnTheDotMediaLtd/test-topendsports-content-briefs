## Brief Phases Explained

1. **Brief Control Sheet** (Phase 1)
   - Research findings
   - Keyword clusters and volumes
   - Competitor analysis
   - Brand selection rationale

2. **Writer Brief** (Phase 2)
   - Content outline with word counts
   - Keyword placement instructions
   - Source requirements
   - Internal linking strategy

3. **AI Enhancement Brief** (Phase 3)
   - Complete HTML structure
   - Schema markup (JSON-LD)
   - Technical implementation details
   - Compliance sections
