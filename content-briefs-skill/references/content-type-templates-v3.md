- Document: Verification date

- **App Store (iOS):** Rating + count, analyze last 30 days
  - Format: "4.9/5 (1.8M reviews) - analyzed Oct 1-14, 2025"
- **Google Play (Android):** Rating + count, recent issues
- **Reddit r/sportsbook:** 10+ threads (20+ upvotes), note consensus

- ‚ùå "October 2025", "Review 2025"

2. Introduction (100-150 words: 40-50 + disclosure)
3. Comparison Table (subject vs 2 competitors)
4. Quick Answer Box
5. [Brand] Overview & Background (300-350w)
6. Getting Started: Sign-Up Process (300-350w)
