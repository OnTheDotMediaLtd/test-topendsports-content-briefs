# Quick Reference Guide - TopEndSports Content Briefs

## Writer Assignment Decision Tree

### URL Pattern to Writer Mapping

**State-Specific Pages** (`/[state-name]/`)
- Writer Assignment: State Expert + Compliance Lead
- Pattern Match: `/pa/`, `/nj/`, `/mi/`, `/il/`, `/va/`, etc.
- Decision: Always verify operator availability for that state
- Note: Requires state-specific research and T&Cs

**Brand Review Pages** (`/[brand-name]-review/` or `/reviews/[brand-name]`)
- Writer Assignment: Brand Specialist
- Pattern Match: `/fanuel-review/`, `/betmgm-review/`, `/reviews/fanduel/`
- Decision: Primary content = brand comparison article
- Note: Brand positioning locked (see Brand Positioning Cheat Sheet below)

**Comparison Pages** (`/vs/`, `/best/[category]`, `/top/[category]`)
- Writer Assignment: Comparison Writer
- Pattern Match: `/fanduel-vs-betmgm/`, `/best-betting-apps/`, `/top-sportsbooks/`
- Decision: Multiple brands featured (8-10 for high-volume keywords)
- Note: Brand cards + comprehensive comparison table required

**How-To Pages** (`/how-to/`, `/guide/`, `/instructions/`)
- Writer Assignment: Technical Writer / How-To Specialist
- Pattern Match: `/how-to-withdraw-funds/`, `/guide-to-mobile-betting/`
- Decision: Step-by-step format required, no brand focus
- Note: Assume user is new to topic; use screenshots

**Sports/League Pages** (`/[sport]/`, `/[league]/`)
- Writer Assignment: Sports Writer + SEO Writer
- Pattern Match: `/nfl/`, `/nba/`, `/mlb/`, `/nhl/`
- Decision: Integrate multiple sportsbooks naturally (avoid hard selling)
- Note: Focus on sport-specific content with operator integration

**Promotional Pages** (`/promo/`, `/bonus/`)
- Writer Assignment: Promotions Specialist
- Pattern Match: `/promo/fanduel-bonus/`, `/bonus-codes/`
- Decision: Time-sensitive; verify current operator offers
- Note: Include expiration dates and terms clearly

**General Informational** (`/news/`, `/tips/`, `/education/`)
- Writer Assignment: Content Generalist
- Pattern Match: `/news/`, `/betting-tips/`, `/odds-education/`
- Decision: Minimal affiliate content; maximum value
- Note: Can integrate operators naturally in dedicated sections

## Template Selection Flowchart

### Decision Path: Content Type Identification

```
START: New Content Assignment
  |
  +---> Is it about a specific brand review?
  |     YES ---> TEMPLATE 1: Brand Review (3500-4000 words)
  |
  +---> Is it comparing multiple brands or "best of" category?
  |     YES ---> TEMPLATE 2: Comparison/Best-Of (2500-3500 words)
  |
  +---> Is it step-by-step instructions (how to do X)?
  |     YES ---> TEMPLATE 3: How-To Guide (1500-2500 words)
  |
  +---> Is it about a specific state/region?
  |     YES ---> TEMPLATE 4: State Page (2000-2800 words)
  |
  +---> ELSE: Informational/Blog Post
        Use hybrid structure (2000-3500 words)
```

### Structure Selection by Main Keyword

**"[Brand] Review" Keywords**
- Template: Brand Review (3500-4000)
- Structure: Last Updated → Affiliate Disclosure → Quick Answer → Comprehensive Sections → Mobile → Bonus → Full T&Cs → FAQ → Schema
- Example: "FanDuel review", "BetMGM sportsbook review"

**"Best [Category]" Keywords**
- Template: Comparison (2500-3500)
- Structure: Intro → Comparison Table → Brand Cards → Methodology → FAQ → Payments
- Example: "Best sportsbooks 2025", "Best betting apps for iOS"

**"[Brand] vs [Brand]" Keywords**
- Template: Comparison (2500-3500)
- Structure: Quick comparison → Head-to-head breakdown → Winner by category → FAQ
- Example: "FanDuel vs DraftKings", "Bet365 vs Caesars"

**"How to [Action]" Keywords**
- Template: How-To Guide (1500-2500)
- Structure: Intro → Step-by-step → Pro tips → Screenshots → FAQ
- Example: "How to sign up for BetMGM", "How to withdraw from FanDuel"

**"[State] Sportsbooks" Keywords**
- Template: State Page (2000-2800)
- Structure: Legal status → Available operators → Bonuses → Retail → T&Cs → FAQ
- Example: "Pennsylvania sportsbooks", "Best New Jersey sports betting apps"

## Keyword Mapping Rules

### Volume-Based Header Hierarchy

**High-Volume Keywords (1000+ monthly searches)**
- Header Level: **H2** (Major sections)
- Keyword Density: 0.8-1.2% of section
- Examples: "FanDuel review", "Best sportsbooks", "How to bet on NFL"
- Treatment: Dedicated full section with multiple subsections
- Minimum Content: 400-600 words in section

**Medium-Volume Keywords (100-1000 monthly searches)**
- Header Level: **H3** (Subsections)
- Keyword Density: 0.5-0.8% of subsection
- Examples: "FanDuel mobile app features", "BetMGM customer service"
- Treatment: Integrated into parent section
- Minimum Content: 150-300 words in subsection

**Long-Tail Keywords (<100 monthly searches)**
- Header Level: **H4** (Sub-subsections)
- Keyword Density: 0.3-0.5%
- Treatment: Bullet points, brief explanations
- Minimum Content: 50-100 words

**Question-Based Keywords**
- Header Level: **FAQ Section** (dedicated)
- Format: "Q: [Full question] A: [150-300 word answer]"
- Minimum Count: 5-10 questions depending on template
- Treatment: Separate FAQ section with Schema markup

### Secondary Keyword Integration Rules

**Cluster Volume Requirement**
- Minimum secondary keywords: 8-15 per article
- Target cluster volume increase: 400-900%
- Secondary keywords must relate to primary keyword topic
- Spread across all section headers (both H2 and H3)

**Placement Strategy**
- First mention: H2 or H3 header (natural inclusion)
- Second mention: Within first paragraph of section
- Remaining mentions: Spread throughout body (1-2 per 100 words)
- Never force keyword; maintain natural readability

**Example Cluster**
Primary: "best sportsbooks" (2,400 monthly volume)
- Secondary 1: "sportsbook apps" (1,100)
- Secondary 2: "betting apps" (950)
- Secondary 3: "online sportsbooks" (850)
- Secondary 4: "mobile betting apps" (720)
- Secondary 5: "sports betting sites" (680)
- Secondary 6: "legal sportsbooks" (620)
- Secondary 7: "top sportsbooks" (580)
- Secondary 8: "sports betting platforms" (450)
Total Cluster: 8,950 monthly volume = 373% increase

## Spanish Content Checklist

- [ ] Use **"ustedes"** for formal address (NOT "vosotros" - Spain Spanish)
- [ ] Never mix Spain Spanish and Latin American Spanish
- [ ] Use **"USTEDES"** conjugation consistently throughout
- [ ] Spell out Spanish accents properly (é, ñ, ü, á, ó, í)
- [ ] Use Latin American Spanish terminology, not Spain-specific

### Age & Legal Compliance

- [ ] Age requirement: **21+** (NOT 18+)
- [ ] Mention age requirement in all promotional sections
- [ ] Use legal terminology appropriate for USA Spanish speakers
- [ ] Include disclaimer: "Disponible solo para mayores de 21 años"
- [ ] Never suggest gambling is available to different age groups

### Operator & Geographic Requirements

- [ ] Reference only **USA-based operators**
- [ ] Do NOT include operators specific to Spain
- [ ] Do NOT include operators specific to Latin America (unless US-licensed)
- [ ] Focus on brands with Spanish-language USA operations
- [ ] Verify operator availability in Spanish-speaking US states

- [ ] Use "apuestas deportivas" (sports betting) - standard term
- [ ] Use "sportsbook" in English context, provide Spanish term
- [ ] Reference USA states and regulations (not Spanish regions)
- [ ] Use USD currency, not EUR
- [ ] Reference English-language app store reviews in Spanish context

### Cultural Considerations

- [ ] Adapt examples to US Hispanic audience preferences
- [ ] Reference US sports primarily (NFL, NBA, MLB, NHL, MLS)
- [ ] Include cultural holidays relevant to USA Hispanic calendar
- [ ] Avoid Spain-specific cultural references
- [ ] Use inclusive language for Latin American heritage

### T&Cs and Legal Language

- [ ] Use formal "usted" in all legal sections
- [ ] Include full legal terminology in Spanish
- [ ] Provide links to Spanish T&Cs pages if available

- [ ] Translate all important legal terms clearly

### Quality Control

- [ ] Proofread for consistency (ustedes throughout)

## Common Mistakes Table

### Content Structure Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|-----------------|
| Wrong word count | SEO performance suffers; reader expectations unmet | Verify template requirement before writing; pad to minimum, trim excess after writing |
| Missing H2 headers | Poor SEO structure; unreadable | Minimum 3 H2 headers; main keyword in first H2 |
| Only 3-4 secondary keywords | Insufficient semantic relevance; low cluster building | Target 8-15 secondary keywords spread across headers |
| FAQ section has <5 questions | Missed schema markup opportunity; low featured snippet potential | Include 5-10 FAQ questions depending on template; every Q needs schema |
| No updated date badge | Reader trust issues; "stale" appearance | Add "Last Updated: [Date]" badge in intro on every review |

### Brand/SEO Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|-----------------|
| FanDuel not in position #1 | Contradicts brand strategy; confuses readers | Always position FanDuel #1 unless explicit testing different angle |
| BetMGM not in position #2 | Breaks brand positioning standard | Always position BetMGM #2 in comparisons |
| Comparing 5-6 brands only | Low word count; incomplete comparison signal | Target 8-10 brands for comparison articles |
| No comparison table | Hard to scan; SEO disadvantage | Include brand comparison table in all "best of" / "vs" content |
| Brand T&Cs incomplete | Legal liability; reader confusion | Include age, wagering, max bonus, time limit, terms URL for EVERY brand |

### Keyword/SEO Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|-----------------|
| Primary keyword never in H2 | Poor SEO signal; lost ranking opportunity | Lead H2 must include or closely match primary keyword |
| Keyword density too high (>2%) | Keyword stuffing penalty | Target 0.8-1.2% for high-volume; 0.5-0.8% for medium |
| No secondary keywords | Insufficient topic coverage; low semantic relevance | Identify 8-15 related keywords; spread across H2/H3 |
| Secondary keywords in unrelated sections | Artificial keyword placement; low relevance signal | Secondary keywords must relate to section topic |
| Long-tail keywords ignored | Lost long-tail traffic opportunity | Include questions and long-tail keywords in FAQ section |

### Compliance/Legal Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|-----------------|
| Age requirement says "18+" | Violates US sports betting age requirement | Always use "21+" |
| No affiliate disclosure | Legal liability; FTC violation risk | Include clear affiliate disclosure near top of content |
| T&Cs section missing | Reader confusion; missing key details | Dedicate full section to T&Cs for main brands reviewed |
| No legal disclaimer | Liability exposure | Include "For entertainment purposes only" or similar at top/bottom |
| Spanish content uses "vosotros" | Not appropriate for USA Spanish audience | Use "ustedes" exclusively in Spanish content |

### Content Quality Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|-----------------|
| No personal experience/voice | Generic, unhelpful content; low engagement | Include specific examples, use first-person perspective |
| Mobile experience not discussed | Incomplete review for modern users (90%+ mobile) | Dedicated subsection: "Mobile App Experience" |
| Bonus analysis shallow | Missed opportunity to add value | Detailed bonus breakdown: bonus amount, wagering requirement, time limit, restrictions |
| Screenshots outdated/wrong | Reader confusion; loss of credibility | Update screenshots with current app versions |
| No pros/cons per brand | Weak comparison content | Include 3-5 pros and cons for each brand reviewed |

### Common Writing Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|-----------------|
| Clickbait headlines | Reader distrust; high bounce rate | Accurate headlines that match content; use number lists where relevant |
| Vague brand claims | Low credibility; legal risk | Back all claims with data, screenshots, or direct testing |
| Too formal/stiff writing | Poor engagement; high bounce rate | Use conversational tone; address reader as "you" |
| No call-to-action | Reader doesn't know next step; lost conversion | Clear CTA: "Click here to sign up" or "Visit FanDuel" in relevant sections |
| Inconsistent terminology | Reader confusion | Define term in first use; maintain consistent naming (not "DK" and "DraftKings" randomly) |

## Secondary Keyword Targets

### Definition & Requirements

**What are Secondary Keywords?**
- Keywords related to main topic
- Lower search volume than primary keyword
- Support semantic relevance and topic authority
- Spread across article sections (H2 and H3 headers)

**Minimum Requirements:**
- Count: 8-15 secondary keywords per article
- Cluster Volume Increase: Target 400-900% increase over primary keyword volume
- Placement: Integrated naturally throughout content
- Relevance: Must relate directly to primary keyword topic

### Secondary Keyword Identification Process

**Step 1: Analyze Primary Keyword**
- Example Primary: "best sportsbooks" (2,400 monthly volume)
- Topic Area: Sportsbook comparison, sports betting apps, online betting

**Step 2: Identify Related Terms**
- Synonyms: "top sportsbooks", "best betting apps", "sportsbook apps"
- Long-tail variations: "best sportsbooks for [sport]", "best mobile sportsbooks"
- Question format: "what is a sportsbook", "how to choose a sportsbook"

**Step 3: Validate Search Volume**
- Minimum per keyword: 100 monthly searches
- Maximum per keyword: Just below primary (to avoid cannibalization)
- Target range: 200-1,500 monthly searches per secondary

**Step 4: Group into Clusters**
- Semantic similarity: Group related terms
- Section mapping: Assign to relevant H2/H3
- Natural integration: Ensure smooth keyword use

**Step 5: Build Cluster**
- List 8-15 keywords with volume data
- Calculate total cluster volume
- Verify 400-900% increase over primary

### Placement Strategy by Content Type

**Brand Review (3500-4000 words):**
- H2 Section Count: 5-6 sections
- H3 Subsection Count: 2-3 per H2
- Secondary Keywords: 8-15 total
- Placement: 1-2 per H2, 1 per H3

**Comparison/Best-Of (2500-3500 words):**
- H2 Section Count: 4-5 sections
- H3 Subsection Count: 1-2 per H2
- Secondary Keywords: 8-12 total
- Placement: 1-2 per H2 section

**How-To Guide (1500-2500 words):**
- H2 Section Count: 3-4 sections
- H3 Subsection Count: 1-2 per H2
- Secondary Keywords: 6-10 total
- Placement: 1 per H2, 1 per H3

**State Page (2000-2800 words):**
- H2 Section Count: 4-5 sections
- H3 Subsection Count: 1-2 per H2
- Secondary Keywords: 8-12 total
- Placement: 1-2 per H2, 1 per H3

### Volume-Based Keyword Distribution

**High-Volume Secondaries (500-1,500 monthly):**
- Count: 3-5 in cluster
- Placement: H2 headers
- Density: 0.7-1.0%
- Importance: High - direct SEO impact

**Medium-Volume Secondaries (200-500 monthly):**
- Count: 4-7 in cluster
- Placement: H3 headers and opening paragraphs
- Density: 0.5-0.8%
- Importance: Medium - topical relevance

**Long-Tail Secondaries (100-200 monthly):**
- Count: 3-6 in cluster
- Placement: FAQ section, bullet points
- Density: 0.3-0.5%
- Importance: Low-medium - long-tail traffic capture

### Common Secondary Keyword Mistakes

| Mistake | Why Wrong | Fix |
|---------|-----------|-----|
| Too few secondaries (3-4) | Insufficient topic coverage | Target minimum 8; aim for 15 |
| Secondaries in wrong sections | Low relevance signal | Map keywords to semantically related sections |
| Cluster volume <300% increase | Insufficient keyword building | Expand secondary list until 400-900% achieved |
| No volume data for secondaries | Can't validate strategy | Every secondary must have search volume |
| Forced keyword placement | Obvious keyword stuffing | Integrate naturally; use synonyms where needed |
| Ignoring long-tail questions | Missing featured snippet opportunity | Include question-based keywords in FAQ section |
