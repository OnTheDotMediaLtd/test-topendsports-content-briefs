## üîç PART 1: BONUS VERIFICATION PROTOCOL

### Critical Rule
**Always verify from brand's official promo page. Never trust third-party sites.**

### 5-Step Verification Process

**Step 1: Find Official Promo Page**
- FanDuel: `sportsbook.fanduel.com/promos`
- DraftKings: `sportsbook.draftkings.com/promos`
- BetMGM: `sports.betmgm.com/promotions`

**Step 2: Copy Exact Bonus Text**
Copy verbatim:
- Headline offer (e.g., "Bet $5, Get $300 in Bonus Bets")
- Visible fine print
- Complete legal disclaimer

**Step 3: Copy Complete T&Cs**
Find and copy:
- Eligibility requirements
- Qualifying conditions
- Bonus terms (amount, timeframe, expiration)
- State restrictions
- Fine print (stake return, excluded markets)

**Step 4: Document Verification**
```
BONUS VERIFICATION LOG
Brand: FanDuel
Offer: "Bet $5, Get $300 in Bonus Bets"
Source: https://sportsbook.fanduel.com/promos
Date Verified: October 14, 2025
Next Due: November 14, 2025
T&Cs: Complete legal language copied
```

**Step 5: Re-Verification Schedule**
- Monthly minimum
- Bi-weekly for high-traffic pages
- Immediately when user reports incorrect bonus
- Before major sports events (bonuses often change)

### State-Specific Verification
Some bonuses vary by state:
- Check for "Offer may vary by state" disclaimer
- Verify each state's specific promo page
- Document state variations separately

### Verification Checklist
Before publishing content with bonuses:
- [ ] Visited brand's official promo page (not affiliate)
- [ ] Copied exact headline offer text
- [ ] Copied complete T&Cs (not summarized)
- [ ] Documented verification date
- [ ] Checked state-specific variations (if state page)
- [ ] Linked to official T&Cs page
- [ ] Set re-verification reminder (monthly)

## üë• PART 2: SOURCE HIERARCHY

### Priority Order

**TIER 1: Direct User Feedback (HIGHEST PRIORITY)**
1. **App Store Reviews (iOS)** - Real customers, verified purchases
   - Format: "4.9/5 (1.8M reviews) - analyzed Oct 1-14, 2025"
   - Citation: "Mentioned in 847 App Store reviews as 'best app interface'"

2. **Google Play Reviews (Android)** - Real customers, verified downloads
   - Format: "4.7/5 (371.9K reviews) - analyzed Sept 15-Oct 14"
   - Citation: "Reported in 402 Google Play reviews (Sept-Oct 2025)"

3. **Reddit r/sportsbook** - Unfiltered opinions, no financial incentive
   - Format: "Reddit users report 24-48 hour payouts (15 threads analyzed)"
   - Search: `site:reddit.com/r/sportsbook [brand name]`

4. **Trustpilot** - Verified customer reviews
5. **Better Business Bureau (BBB)** - Official complaint database

**TIER 2: Brand Official Sources (Verification Only)**
- Brand's website: Feature verification only
- Brand's social media: Announcements, issue acknowledgments

**TIER 3: Industry/Regulatory Sources (Facts Only)**
- State Gaming Commissions: Legal status, licenses
- Industry Publications: Market data, revenue figures

**TIER 4: Affiliate Sites (Use Sparingly)**
- ‚ö†Ô∏è Research gaps only
- ‚ùå NEVER cite for pros/cons or user experience

### Citation Format

‚úÖ **Good Citation:**
```
"FanDuel's mobile app receives consistent praise from users, with 847 App Store
reviewers specifically mentioning 'best app interface' in reviews from the last
30 days (4.9/5 rating across 1.8M total reviews, analyzed Oct 1-14, 2025)."
```

‚ùå **Bad Citation:**
```
"FanDuel has a great mobile app." (No source)
```

### Source Documentation Format
```
SOURCE: App Store Reviews
TYPE: iOS App Store
DATE ACCESSED: October 14, 2025
DATA POINT: 4.9/5 rating (1.8M reviews)
FINDING: "Best app interface" (847 mentions in last 30 days)
EXAMPLE: "FanDuel's interface is cleanest compared to DK/MGM" (Oct 12, 23 helpful votes)
```

## üìÖ PART 3: DATING LANGUAGE RULES

### NEVER Use in Titles/H1s
‚ùå Forbidden:
- Month + Year: "October 2025", "Oct 2025"
- Year alone: "Best Sites 2025", "Review 2025"
- Time-sensitive: "Latest", "Updated for 2025", "Current"

‚úÖ Use Instead:
- Status/Position: "The #1 Rated App", "Industry-Leading Odds"
- Comprehensive: "Complete Guide", "In-Depth Analysis"
- Value Props: "Best for NFL Props", "Fastest Payouts"

### WHERE Dates ARE Appropriate

**‚úÖ "Last Updated" Badge** (immediately after H1):
```html
<div style="background: #e8f5e9; padding: 0.75rem 1.25rem; border-left: 4px solid #2e7d32; margin-bottom: 1.5rem; border-radius: 4px;">
  <p style="margin: 0; font-size: 14px; color: #2e7d32;">
    <strong>‚úì Last Updated:</strong> October 14, 2025
  </p>
</div>
```

**‚úÖ Other Appropriate Places:**
- Schema `dateModified` field
- Within content: "As of October 2025..."
- T&Cs verification: "Last Verified: October 14, 2025"
- User feedback ranges: "Reviews analyzed Oct 1-14, 2025"

### Examples: Before & After

**‚ùå BEFORE (Dated):**
```
H1: FanDuel Review October 2025: Expert Analysis
```

**‚úÖ AFTER (Evergreen):**
```
H1: FanDuel Sportsbook Review: The #1 Rated Betting App
[Last Updated: October 14, 2025 badge]
```

### Two Versions of T&Cs

**1. Brief T&Cs (Inline)**
Use in: Comparison tables, Quick Answer Boxes, callouts
```

```

**2. Complete T&Cs (Expanded)**
Use in: Dedicated bonus sections
Must include:
- Exact bonus text from brand site (verbatim)
- All eligibility requirements
- All qualification steps
- All bonus terms and restrictions
- Complete legal language (not summarized)
- Link to official terms
- Last verified date
- Problem gambling resources

   - If no ‚Üí Remove dated language from body

   - If unsure ‚Üí Re-verify before publishing

   - If affiliate sites ‚Üí Find real user sources

   - If no ‚Üí Get exact language from official source

   - If no ‚Üí Add source or remove claim
