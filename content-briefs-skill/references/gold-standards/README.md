# Gold Standard Templates

**Purpose:** Reference examples that demonstrate V3 brief quality standards.

**Usage:** When generating a new brief, reference the appropriate gold standard for structure and detail level.

---

## Available Gold Standards

| Page Type | File | Lines | Key Features |
|-----------|------|-------|--------------|
| Hub Page | `gold-standard-hub-brief.md` | ~1,500 | Anti-cannibalization, 75-100w mobile sections |
| Comparison Page | `gold-standard-comparison-brief.md` | ~1,200 | Full brand comparisons, 100-150w mobile |
| Review Page | `gold-standard-review-brief.md` | ~800 | Single brand focus, detailed sections |

---

## V3 Requirements Checklist

Every gold standard demonstrates:

### Structure
- [ ] Word Count by Section table at top
- [ ] Competitor URLs section with 2-3 references
- [ ] "Writer must cover" bullets under each H2/H3
- [ ] Internal links mapped to specific sections

### Keywords
- [ ] Total Secondary Volume sum calculated
- [ ] "Unmapped Keywords: NONE" verification
- [ ] Cluster increase % over primary

### Compliance
- [ ] E-E-A-T author requirements
- [ ] Market-specific responsible gambling
- [ ] No affiliate disclosure (sidebar handles it)

---

## How to Use

1. **Identify page type** (hub, comparison, review)
2. **Open corresponding gold standard**
3. **Copy structure** for new brief
4. **Adapt content** to target market/topic
5. **Verify V3 checklist** before delivery

---

## Updating Gold Standards

When improving gold standards:
1. Make changes to the gold standard file
2. Update the line count in this README
3. Document changes in `lessons-learned.md`
4. Update the V3 validation script if needed
